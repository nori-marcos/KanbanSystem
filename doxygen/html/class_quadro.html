Detailed Description

Classe que representa o quadro de um usuário. O quadro é usado para organizar os cartões. Regras de formato:

  • Código: "AB12".
  • Nome: "Nome do quadro".
  • Descrição: "Descrição do quadro".
  • Limite: 5, 10, 15 ou 20.
